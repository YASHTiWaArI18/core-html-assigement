1. Html

    Primary Use Case Key Features/Benefits
    Building the structure of web pages
    • Simple and easy to learn
    • Compatible with all web browsers
    • Allows for semantic markup

  2. CSS

    Primary Use Case Key Features/Benefits
    Applying styles to HTML elements
    • Simple and easy to learn
    • Can be used in conjunction with JavaScript or as a standalone language.
    • Provides control over how content is displayed on the page.

    3. Java Script

    Primary Use Case Key Features/Benefits
    Adding interactivity to websites using client-side scripting.
    • Simple and easy to learn
    • Can be executed by any web browser, making it accessible to users without server-side programming.
    • Can be executed by any web browser without requiring a server connection.
